运算符

算数运算符：

%–取余运算

1、结果的符号与被余数的符号相同

2、开发中经常用来判断能否被整除的情况

eg: int m1=-12,m2=5；

m1%m2=-2；

=–赋值运算

1、可连续赋值，j=k=1；

2、复合赋值（-= += *= /=等）不会改变变量本身的数据类型

short s1 = 10

s1 = s1+2//编译失败（2为int s1为short）

s1+= 2 //编译成功且s1仍然为short

instanceof–检查是否是类的对象

要么是true，要么是false

位运算符位

(操作符操作的都是整型的数据)

  1. << ：在一定范围内，每向左移一位，相当于*2（空缺位补0）

  2. >> : 在一定范围内，每向右移一位，相当于/2（首位复制，空缺补0）

  3. >>> : 无符号右移(不管正负缺位都补零，负数经该操作会变成正数）

  4. 如 int i=21;

    System.out.print(i<<2); 212\2=84;

    System.out.print(i<<3); 212\2*2;

    高效计算 2*8

    2<<3 or 8<<1

    5.^ :相同2进制结果是0 1^1=0,不同二进制结果是1 0^1=1;

    6.~ : 正数取反，各二进制按补码各位取反 ；负数取反，各二进制按补码位取反

取反 ：包括最高位（符号位)0变1；1变0。

（正数的取反为补码，负数取反后得到的数值要 + 1）
