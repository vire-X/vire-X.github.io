JAVA数组

1、是多个相同类型按一定顺序排列的集合，并使用一个名字命名，并通过编号的方式对这些数据进行统一管理。

–>数组名

–>元素（有序排列）

–>下标或角标、索引（从0开始数）

–>数组的长度（元素的个数）

2、数组属于引用类型的变量，而数组中的元素可以是任何数据类型，包括基本数据类型和引用数据类型

3、创建数组对象就会在内存中开辟一整块连续空间，而数组名中引用的是这块连续空间的首地址

4、数组的长度一旦确定就不能修改

5、如何获得数组的长度// 属性：length

一维数组的使用

1、数组声明及初始化
JAVA中数组会自动初始化

–>整型数组元素的默认初始化值为 ： 0

–>浮点数组元素的默认初始化值为 ： 0.0

–>char数组元素的默认初始化值为 ：0（ascii值为0 的元素效果类似于空格，非‘0’）

–>Boolean型元素默认为： false

–>数组类型为引用数据类型：null （意为空值）
