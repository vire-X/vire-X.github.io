java—面向对象（类与方法）

面向对象的三条主线

1、java类及类的成员：属性、方法、构造器；代码块、内部类

2、面向对象的三大特征：封装性、继承性、多态性（、抽象性）

3、其他关键字：如 this、super、static、final、abstract等

类和对象都是面对对象的核心概念
类 是对一类事物的描述，是抽象的、概念至上的定义

对象 是实际存在的该类事物的每个个体，因而也称为实例

面向程序设计的重点其实就是类的设计

设计类，就是设计类的成员

类和对象的使用

1、创建类，设计类的成员

2、创建类的对象

3、通过 “ 对象 . 属性 “ 或 “ 对象 . 方法 ” 调用对象的结构

Field = 属性 = 成员变量，Method = （成员）方法 = 函数

创建类的对象 = 类的实例化 = 实例化类

如果创建了一个类的多个对象，则每一个对象都独立拥有一套类的属性。（非static的）

类中属性的使用

属性（成员变量） vs 局部变量

  • 1、相同点：
  • 1.1定义变量的格式：数据类型 变量名 = 变量值
  • 1.2先声明,后使用
  • 1.3变量都有其对应的作用域
  • 2、不同点：
  • 2.1在类的声明位置不同
    • 属性：直接定义在类中的一对{ }内
    • 局部变量：声明在方法内、方法形参、代码块内、构造器内、构造器形参的变量
  • 2.2关于权限修饰符的不同
    • 属性 ：可以在声明属性的时候 , 指明其权限、使用权限修饰符
    • 常用的权限修饰符：privat、public、缺省、protected—>封装性
    • 局部变量：不可以使用权限修饰符
  • 2.3默认初始化值的情况
    • 属性：类的属性，根据其类型，都有默认初始化值
    • 整型（byte、short、int、long）：0
    • 浮点型（float、double）：0.0
    • 字符型（char）：0 （或’ \u0000 ‘）
    • 布尔型（Boolean）：false
    • 引用数据类型（类、数组、接口）：null
    • 局部变量：没有默认初始化值
    • 意味着，我们在调用局部变量之前，一定要显式赋值
    • 特别的，形参在调用时，
  • 2.4在内存中的位置
    • 属性：加载在堆空间中
    • 局部变量：加载在栈空间中

类中方法的声明和使用

  • 方法：描述类应该具有的功能（类似于函数）

  • 比如：Math类，sqrt类\random() .…

  • Scanner类 nextXxx()…

  • Arrays类：sort() \ binarySearch() \toString() \equals()…

  • 1.方法的声明： //不可以给在定义形参时给形参赋值

  • ​ 权限修饰符 返回值类型 方法名(形参列表){

  • ​ 方法体 //方法其实类似于c语言中的函数

  • }

  • 举例：

  • public void eat(){…}

  • public void sleep(int hour){…}

  • public String getName(){…}

  • public String getNation(String nation){…}

  • 注意：static、final、abstract 来修饰方法

  • 3.说明：

  • 3.1 关于权限修饰符：(封装性)

  • java中四种权限修饰符：

  • private、public、缺省、protected

  • 3.2 返回值类型：有返回值、无返回值

  • ​ 1、如果方法有返回值：则必须在方法声明是指定返回值的类型

  • 同时，方法中需要使用return来返回指定类型的数据。

  • 如果无返回值:则方法声明时使用void来表示，并且不带return

  • 如果使用的话，只能用“return”; 表示方法结束的意思。

  • ​ 2、方法是否需要返回值

  • ​ 依照题目要求和经验分析

  • 3.3 方法名：属于标识符。

  • 标识符的规范规则：见名知意

  • 3.4 方法可以声明任意个形参

  • 格式：数据类型 形参 , 数据类型 形参, …..

  • 3.5 方法体：方法功能的体现**

  • 4.return关键字的使用

  • 1.使用范围：方法体中

  • 2.作用： 结束方法

  • 针对有返回值的方法，用 return来返回数据

  • 3.注意点：return关键字后不可以声明执行语句。

  • 5.方法的使用中可以调用当前类中的属性和方法

  • 特殊：方法A中又调用了A方法–> 递归方法

  • 方法中不可以定义方法
